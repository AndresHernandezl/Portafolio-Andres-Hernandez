Instalar Sublime Text En Debian

Sublime Text puede ser utilizado de forma gratuita pero la versión de evaluación te mostrará un alerta o aviso la Quinta vez que guardes algún archivo después de abrir el programa y luego no verás ese alerta hasta la próxima vez que uses el editor. Pero actualmente no hay ningún límite de tiempo para la versión de evaluación,por ello podemos utilizarla cuando quieras.Si tienes Debian Linux y deseas utilizar este editor, entonces vamos aprender como hacerlo.
